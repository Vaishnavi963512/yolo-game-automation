# 🎮 YOLO Game Automation System

This project is a Computer Vision-based automation system that uses YOLO and OpenCV to detect objects in a game environment and automate data collection.

---

## 🚀 Features

- Real-time object detection using YOLO  
- Automatic screenshot capturing  
- Auto-labeling of detected objects  
- Efficient dataset generation for training ML models  
- Works on live game environments  

---

## 🛠️ Tech Stack

- Python  
- YOLO (You Only Look Once)  
- OpenCV  
- NumPy  

---

## 📂 Project Overview

This system captures game frames, detects objects using YOLO, and automatically labels them. It helps in reducing manual effort in dataset creation and enables faster model training.
